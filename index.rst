CAPP Camp: Software Setup
=========================

This site contains instructions for setting up the software that you
will use during CAPP Camp and in your fall Computer Science classes

- **Secure Shell (SSH)**  This section includes instructions on how to access the CS Department's Linux servers, including when working off-campus.
- **Visual Studio Code**: Instructions on how to install and configure Visual Studio Code, a popular code editor.
- **GitHub** Instructions for how to get set up to use Git, a popular version control tool, and GitHub, a popular hosting service for Git.

This document contains customized and updated versions of the material
in the `UChicago CS Student Resource Guide
<https://uchicago-cs.github.io/student-resource-guide/>`__.  Many
people have contributed the CS Student Resource Guide over the years,
including Borja Sotomayor, James Turk, Anne Rogers, and numerous other
instructors and teaching assistants over the years.
